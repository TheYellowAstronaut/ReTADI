# ReTADI - <ins>Re</ins>purposed <ins>T</ins>ablet <ins>A</ins>s <ins>D</ins>esk <ins>I</ins>nterface

An **[open source*](#license)** and **web-based** software with a **Rust-powered** server for transforming your old Tablet device to a useful Desk companion.

## Installation

There are a few ways to install ReTADI server. The tablet (client device) doesn't require installation of software (except a web browser, ofc).

### 1] Download installers and pre-built binaries. (Recommended)

Visit our website to download them. [Visit↗]

OR

Download them via the [releases tab](https://github.com/theyellowastronaut/retadi/releases) of this repository.

### 2] Build from source

> Note: You need to install **rust** to build the ReTADI Server. [Download Rust↗](https://www.rust-lang.org)

#### i] Cloning and building

```
git clone https://github.com/TheYellowAstronaut/ReTADI.git
cd ReTADI/source
cargo build
```

#### ii] Running

```
cargo run
```
